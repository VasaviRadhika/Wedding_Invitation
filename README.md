# Project Overview

## Title: Wedding Invitation Management System

### Introduction

The Wedding Invitation Management System is a digital platform designed to simplify the process of creating, managing, and distributing wedding invitations. Traditional invitation methods often involve significant time, cost, and manual effort. This project provides an efficient solution by enabling users to design customized invitations, manage guest lists, send invitations electronically, and track RSVP responses.

### Objective

The primary objective of this project is to develop a user-friendly system that automates wedding invitation management and enhances communication between hosts and guests.

### Scope of the Project

The system allows users to:

* Create and customize wedding invitations.
* Manage guest information and contact details.
* Send invitations through digital channels.
* Track RSVP responses in real time.
* Store event details securely.
* Generate reports on guest attendance and invitation status.

# Wedding Invitation Website

## Project Overview

This project is a simple and elegant Wedding Invitation Website developed using HTML and CSS. The website serves as a digital wedding invitation that provides guests with important event details such as the couple's names, wedding date, venue, schedule, and contact information.

The main objective of this project is to create an attractive, responsive, and user-friendly wedding invitation webpage that can be easily shared online. The design focuses on aesthetics, readability, and accessibility while offering a modern alternative to traditional printed invitations.

## Features

* Elegant and responsive design
* Wedding event details section
* Bride and groom introduction
* Venue and schedule information
* Photo gallery section
* Contact information
* Mobile-friendly layout

## Technologies Used

* HTML5
* CSS3

## Project Structure

* `index.html` – Main webpage structure
* `style.css` – Styling and layout design


## Purpose

The project demonstrates the use of HTML and CSS for creating a visually appealing static website. It showcases web design principles such as responsive layouts, typography, color schemes, and user interface design.
